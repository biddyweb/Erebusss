Erebus is an Open Source RPG (Role-Playing Game), for PCs, smartphones, tablets and handhelds.

This is currently under development and in its early stages, though already playable with three quests, as well as having randomly generated dungeons to explore.

Video of gameplay (v0.5)

Features:

  • Classic point-n-click style RPG, with dungeons to explore, enemies to fight, NPCs to talk to, sub-quests to complete, scenery to interact with, weapons, treasure and other items to find.
  • Also supports Rogue-like keyboard controls.
  • Multiple quests (currently three, more will be added as development progresses!)
  • Also provides randomly generated dungeons to explore.
  • Choice of starting characters (currently Barbarian, Elf, Halfling, Ranger, Warrior), each with their own special skills.
  • Start straight into the action - none of this "For your first quest, please find your next door neighbour's pet cat".
  • Vector-based world rather than tile-based - so items/scenery can be placed in any position, or aligned in any direction.
  • 2D animated graphics, with zoom in/out, and lighting effects.
  • Completely free and Open Source - no ads, unlike many free Android apps.
  • User interface optimised to work with mouse, keyboard and/or touchscreen.
  • Cross-platform - available for Windows, Linux, Android and Pandora. Older versions are also available for Nokia Symbian smartphones.

Download

Erebus is available for Windows, Linux, Android and Pandora. Older versions were also available for Nokia Symbian.

  • For Windows, download the installer. If you get an error when trying to run, you may need to install the Visual C++ 2008 Redistributable. Erebus is tested on Windows 7 and 8. Erebus can be controlled with keyboard and/or mouse, and also supports Windows 8 touchscreens/tablets (tested on the Asus Transformer Book T100). Note that Windows RT (ARM) is not supported, only "full" x86 Windows. On Windows XP, the game may crash on exit - this seems to be due to a bug with SFML. Windows Vista onwards should be fine.

  • The source code is available, released under the GPL v3 or later. For Linux, you need to compile from the source. Please see the file erebus_source.txt (in the source archive) for details on how to compile. Future versions will have binaries available for Ubuntu. There is also a "portable" binary version available for 32-bit Linux at PortableLinuxGames (though this may be an older version). Erebus has been tested on Ubuntu 14.04 64-bit.

  • For Android smartphones/tablets, download from Google Play. Erebus requires Android 2.3.3 or higher, and has been tested with the Samsung Galaxy Nexus running Android 4.3 and Asus Nexus 7 (2013) running Android 4.4. A resolution of at least 800x480 is recommended.

  • For the Open Pandora, download the binary package. You can also compile from source (same instructions as for Linux).

  • For Nokia Symbian smartphones, download from Opera Mobile. Note that this is not the latest version, but v0.10 (the Symbian port has been dropped, as Nokia no longer allow Symbian updates to the Nokia Store - it may still be possible to compile the latest version from the source, but this is untested.) This requires S60 5th edition (Symbian^1) or later - the Nokia Store should let you know if it's compatible with your phone. Erebus has been tested on the Nokia 5800. Note the download count only shows those since migrating to Opera in 2015; the number of Symbian downloads on Nokia store was well over 100,000!

  • A full list of download files, including older versions, is at https://launchpad.net/erebus/+download.

If you are upgrading from a previous version, the new version should be compatible with old save game files, but for best results please start with a new game.
